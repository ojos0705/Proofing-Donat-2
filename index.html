ℹ️ Tentang Aplikasi

🍩 Apa itu Donut Proofing Agent?

Aplikasi asisten berbasis pintar yang dirancang khusus untuk membantu pembuat donat (*bakers*) menentukan waktu fermentasi (*proofing*) adonan secara presisi berdasarkan kondisi suhu dan kelembaban lingkungan sekitar secara real-time.

🧠 Bagaimana AI Bekerja?

Aplikasi mendeteksi parameter cuaca lokal Anda melalui integrasi GPS/IP Jaringan. Menggunakan rumus adaptasi termal, Agen AI akan memotong atau memperpanjang waktu *proofing* dasar secara otomatis. Jika suhu ruangan panas, waktu diperpendek untuk menghindari *over-proofing*. Sebaliknya, jika dingin, waktu akan disesuaikan agar ragi mengembang sempurna.

👑 Fitur Utama

  • Sinkronisasi Cuaca Otomatis: Mengambil data suhu, kelembaban, dan tekanan udara secara langsung.
  • 3 Mode Cepat: Pengaturan instan untuk tahap *Initial*, *Rounding*, dan *Final Proofing*.
  • Mode Luring Premium: Tetap berfungsi di dapur tanpa koneksi internet dengan input parameter manual.
  • Sistem Peringatan Aktif: Dilengkapi alarm suara dan getaran real-time agar adonan tidak rusak.

👥 Tim Pengembang

Dikembangkan dengan penuh dedikasi untuk memajukan kualitas produksi roti dan donat rumahan maupun profesional melalui pendekatan teknologi cerdas.
